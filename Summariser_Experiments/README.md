# DSA4213-Summariser-Grp30
This folder contains the experimental fine-tuned models for each component of the Summariser Model, followed by the various Jupyter notebooks used for training, evaluation and debugging of code. 

### If you want to reproduce the evaluation results (skip the training/fine-tuning cells as instructed within each notebook), ensure the following setup:
Apart from the files/folders within this sub-folder, also include:
  - Event-summariser-LoRA-v1
  - Ending-summariser-LoRA-v2
  - QA-loRA-v2
    
by copying them over from the main repo folder. Then, make sure you have followed the instructions in the main folder's README.
